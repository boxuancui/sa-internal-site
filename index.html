AWS S3

S3© is a cheap object storage service offered by Amazon. It is particularly useful when loading data to Redshift. You will need the access key ID and access key to connect to BCG S3 buckets.

Required Knowledge: mouse and keyboard

  • Relevant Links: Request BCG S3 Access,  BCG AWS Console,  CloudBerry Explorer

AWS EC2

EC2© is a scalable cloud server with elastic pricing. The EC2 instance for Strategic Analytics is a VPC (Virtual Private Cloud) behind BCG firewall. Currently, it is used to host services like this webpage, shiny apps and other internal services.

Required Knowledge: Linux

  • Relevant Links: PuTTY for SSH,  SA HTTP Server,  SA RStudio Server,  SA Shiny Server
  • Documentation: Access SA EC2,  Linux Beginners Guide,  Linux Admin Guide

AWS Redshift

Redshift© is a fast and scalable data warehouse solution. BCG has its own architecture to access due to various security reasons.

Required Knowledge: SQL

  • Relevant Links: Request Redshift Service
  • Documentation: Official Reference Guide,  Redshift Pricing

RStudio Server

R is installed on the cloud server for Strategic Analytics. You may use R collaborately with any member of the team through RStudio Server©.

Required Knowledge: R

  • Relevant Links: SA RStudio Server
  • Documentation: How to access?,  Admin Guide 101,  Admin Guide 999

Shiny App

Shiny© turns your analyses into interactive web applications by compiling R code into HTML and JavaScript.

Required Knowledge: R

Existing Apps: Lunch Buddy,  Bo's Little Text Miner,  Top Populated Cities,  Store Performance,  Orkin Dashboard,  TEDx Radar

  • Relevant Links: Official Site,  Demo Apps,  Getting Started,  Becoming Expert
  • Documentation: Deploy Shiny Apps in SA

GitHub

GitHub© is a web hosting service for Git repositories. BCG has a GitHub Enterprise© account and is accessible to all BCG employees who needs distributed revision control and source code management functionality.

Required Knowledge: Git, Basic Command Line (Linux Bash)

  • Relevant Links: BCG GitHub,  SA GitHub,  GitHub Navigator Page
  • Documentation: SA GitHub Overview Official Documentation 5-min Guide Git 101
